# AWS CLI Generator

自然言語の指示から AWS CLI コマンドを生成するスキルです。

## 入力

自然言語での操作指示。例:
- 「prod クラスターの api サービスを停止したい」
- 「S3 バケット my-bucket のオブジェクト一覧を取得」
- 「RDS インスタンスのスナップショットを作成」

引数がない場合は「どのような AWS 操作をしたいか教えてください」と聞く。

## 対応サービス

- **ECS**: サービス更新・タスク停止・デプロイ確認
- **EC2**: インスタンス操作・セキュリティグループ・AMI
- **S3**: オブジェクト操作・バケット設定
- **RDS**: スナップショット・パラメータ確認・フェイルオーバー
- **CloudWatch**: ログ取得・メトリクス確認・アラーム操作
- **IAM**: ポリシー確認・ロール操作
- **SSM**: パラメータ取得・セッションマネージャー接続
- **ElastiCache**: クラスター情報・フェイルオーバー
- **Lambda**: 関数呼び出し・ログ確認
- **ECR**: イメージ一覧・認証トークン取得

## 出力フォーマット

```
## AWS CLI Command

\`\`\`bash
aws <service> <command> \
  --option1 value1 \
  --option2 value2
\`\`\`

### 説明
(コマンドが何をするかの説明)

### 注意
(実行前に確認すべきこと、副作用など)

### 関連コマンド
\`\`\`bash
# (確認・ロールバック等に使えるコマンド)
\`\`\`
```

## ルール

- 破壊的操作 (削除・停止・変更) には必ず `--dry-run` オプションがある場合はそれも示す
- 複数ステップが必要な場合はスクリプト形式で提示する
- リージョンが不明な場合は `--region ap-northeast-1` をデフォルトとして使用
- プロファイル指定が必要な場合は `--profile <profile>` をコメントで示す
- 出力は日本語
